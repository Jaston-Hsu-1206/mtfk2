# Rule: 山羌特調不可無糖 (`muntjac_special_sugar_rule`)

## 規則描述 (Rule Description)

此規則旨在確保山羌茶坊招牌「山羌特調」系列的獨特山林風味。由於特調飲品的基底本身就加入了我們特製的天然蜂蜜，因此無法製作完全「無糖」的選項，最低甜度限制為「微糖」。

## 觸發條件 (Trigger Conditions)

當顧客點選的飲料滿足以下兩個條件時觸發此規則：

1.  飲料的品項名稱包含「山羌特調」（例如：山羌特調奶茶、山羌特調鮮奶）。
2.  顧客要求的甜度屬性為「無糖」。

## 執行動作 (Action on Trigger)

1.  **拒絕無糖設定**: 系統自動阻止將該「山羌特調」飲品的甜度設為無糖，最低只能選擇微糖。
2.  **提示 Agent**: Agent（山羌茶坊店小二）應以活潑的語氣，禮貌地告知顧客此限制，並解釋原因，同時建議可選擇的最低甜度。

## 範例 (Examples)

| 顧客點餐 | 規則判斷 | Agent 行為 |
| :--- | :--- | :--- |
| 「我要一杯山羌特調奶茶，無糖去冰。」 | 觸發 | 「汪！不好意思喔，我們的山羌特調本身就有加入天然蜂蜜提味，沒辦法做完全無糖呢！幫您做最低甜度的『微糖』好不好呀？」 |
| 「一杯秘境高山青茶，無糖。」 | 不觸發 | 正常處理訂單（高山青茶不含特調蜂蜜基底，可做無糖） |
| 「一杯山羌特調奶茶，半糖少冰。」 | 不觸發 | 正常處理訂單 |

## 影響 (Impact)

*   **風味保證**: 維護山羌茶坊招牌飲品的最佳口感與獨特風味。
*   **顧客滿意度**: 透過親切的解釋，讓顧客理解並接受產品特色，避免誤會。

## 備註 (Notes)

此規則為系統硬性限制，Agent 無法繞過。這也是展現山羌茶坊對天然食材堅持的一環。
